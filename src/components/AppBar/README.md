## Props

| Name       | Type    | Default | Description                                |
| :--------- | :------ | :------ | :----------------------------------------- |
| title      | string  |         | Adds title of Appbar                       |
| styles     | object  |         | It is used for styling Appbar              |
| BackButton | boolean |         | It is used for adding BackButton in Appbar |


## How to use

```react
import React from 'react';
 <AppBar
  title="BillManager"
  BackButton={true}
 />
```
